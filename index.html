⚙️ Panduan Setup Database Terpusat

Ikuti langkah berikut untuk menghubungkan dashboard ini ke Google Sheets sebagai database bersama. Gratis, mudah, tidak perlu coding.

📊 LANGKAH 1 — Buat Google Sheets

Ini adalah database tempat semua data tersimpan

1
Buka Google Sheets Pergi ke sheets.google.com → klik "+ Blank" untuk buat spreadsheet baru
2
Beri nama file Klik "Untitled spreadsheet" di pojok kiri atas → ganti menjadi KPI Marketing 2026
3
Buat header di baris 1 Ketikkan header berikut di baris pertama (tiap kata di kolom berbeda): A1: timestamp | B1: bulan | C1: leads_kss | D1: leads_kti | E1: opp_kss | F1: opp_kti | G1: event_leads | H1: strategic_partner | I1: views_total | J1: engagement_total | K1: followers_total | L1: npi_sales | M1: input_by
4
Beri nama sheet Klik kanan tab "Sheet1" di bawah → pilih "Rename" → ketik Data

🔧 LANGKAH 2 — Buat Apps Script

Hanya untuk SIMPAN data (POST). Baca data pakai CSV — tidak perlu doGet.

1
Buka Apps Script Di Google Sheets, klik menu Extensions → Apps Script
2
Hapus kode yang ada Pilih semua teks di editor (Ctrl+A) lalu hapus
3
Copy & paste kode ini:
// Apps Script hanya dipakai untuk WRITE (POST) // READ dilakukan via Google Sheets CSV — tidak perlu doGet lagi const SHEET_NAME = 'Data'; function doPost(e) { try { const body = JSON.parse(e.postData.contents); const sheet = SpreadsheetApp.getActiveSpreadsheet() .getSheetByName(SHEET_NAME); const row = [ new Date(), body.bulan, body.leads_kss, body.leads_kti, body.opp_kss, body.opp_kti, body.event_leads, body.strategic_partner, body.views_total, body.engagement_total, body.followers_total, body.npi_sales, body.input_by ]; const existing = sheet.getDataRange().getValues(); for (let i = 1; i < existing.length; i++) { if (String(existing[i][1]) === String(body.bulan)) { sheet.getRange(i+1, 1, 1, row.length).setValues([row]); return ok('updated'); } } sheet.appendRow(row); return ok('saved'); } catch(err) { return ContentService .createTextOutput(JSON.stringify({ok:false,error:err.message})) .setMimeType(ContentService.MimeType.JSON); } } function ok(msg) { return ContentService .createTextOutput(JSON.stringify({ok:true,msg})) .setMimeType(ContentService.MimeType.JSON); }

🚀 LANGKAH 3 — Deploy Web App

Buat URL publik untuk dashboard terhubung

1
Simpan script Klik ikon 💾 save atau tekan Ctrl+S
2
Klik Deploy Di pojok kanan atas, klik tombol biru "Deploy" → pilih "New deployment"
3
Isi pengaturan deploy: • Type: Web App
• Description: KPI Dashboard
• Execute as: Me
• Who has access: Anyone ← penting!
4
Klik "Deploy" → "Authorize access" → pilih akun Google Anda → klik Allow
5
Copy Web App URL Akan muncul URL yang diawali https://script.google.com/macros/s/... — copy URL ini
💡 Setiap kali Anda edit script, klik Deploy → Manage deployments → Edit → pilih "New version" agar perubahan aktif.

📡 LANGKAH 3B — Publish Sheets ke CSV

Ini yang dipakai dashboard untuk MEMBACA data — bebas CORS

1
Buka Google Sheets → klik menu File → Share → Publish to web
2
Pilih sheet "Data" di dropdown pertama → pilih Comma-separated values (.csv) di dropdown kedua
3
Klik Publish → konfirmasi OK → copy link CSV yang muncul
4
Link CSV terlihat seperti ini: https://docs.google.com/spreadsheets/d/ID_SPREADSHEET/pub?gid=0&single=true&output=csv
⚠️ Ini BERBEDA dari link spreadsheet biasa. Harus dari menu Publish to web, bukan copy dari address bar.

🔗 LANGKAH 4 — Hubungkan Dashboard

Masukkan URL ke dashboard ini

1
Klik tombol status di pojok kanan atas dashboard (bertuliskan "Belum terhubung")
2
Paste URL yang tadi di-copy dari Apps Script → klik "Hubungkan & Mulai"
3
Bagikan file HTML ini ke seluruh anggota tim melalui Google Drive atau email. Mereka masing-masing hanya perlu memasukkan URL yang sama.
4
Selesai! Setiap tim input data → otomatis tersimpan di Google Sheets bersama → semua orang lihat data yang sama saat klik 🔄 Refresh
📌 Tip untuk Notion: Buka Notion → ketik /embed → paste URL file HTML yang sudah di-host di Google Sites atau GitHub Pages. Panduan hosting ada di bawah.
